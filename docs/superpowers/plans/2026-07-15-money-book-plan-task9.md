# Task 9: 设置页详细实现

## SettingsPage.vue

设置页是一个长表单页面，按 iOS 设置风格分组。

### 数据管理
- **管理账户**：跳转子页面 AccountManagement.vue（或弹窗），列表显示所有账户 → 点击编辑 → 弹窗修改名称/图标/排序 → 左滑或点击删除（有流水弹窗提示"该账户已有流水记录，无法删除"）
- **管理分类**：跳转子页面 CategoryManagement.vue，显示一级分类列表 → 展开二级 → 切换显示/隐藏
- **管理标签**：简单标签列表 + 新增输入框 + 删除按钮

### 周期记账
- **管理周期规则**：跳转子页面 RecurringRules.vue，列表显示所有规则 → 新增 → 编辑 → 启用/停用开关 → 删除
- 规则表单：类型、金额、账户、分类、标签、备注、每月第几日、启用状态

### 功能组

```
  关于钱书           v1.0.0
```

**PIN 码逻辑:**
- 当前无 PIN：弹窗要求输入新 PIN 两次 → 哈希后存入 localStorage
- 当前有 PIN：先验证旧 PIN → 再输入新 PIN 两次 → 更新
- 关闭 PIN：验证当前 PIN → 清除 localStorage

**彻底销毁确认流程:**
1. 点击"彻底销毁数据"
2. 弹窗确认："此操作将清除所有数据，且无法恢复。确定继续？"
3. 再次确认："请再次确认"
4. 执行 destroyAllData()
5. 提示成功，建议刷新页面
